- 💻 **源代码**：[github.com/G4brym/R2-Explorer](https://github.com/G4brym/R2-Explorer)

## 概述

jc-r2-explorer 通过现代化、用户友好的界面改变了您的 Cloudflare R2 存储体验。它提供强大的文件管理功能，同时依托 Cloudflare 的基础设施保持企业级安全性。

## 主要特性

- **🔒 安全性**
  - 支持基础认证（Basic Authentication）
  - Cloudflare Access 集成
  - 在您的 Cloudflare 账户上自托管

- **📁 文件管理**
  - 拖拽上传文件
  - 创建和组织文件夹
  - 大文件分片上传
  - 右键上下文菜单提供高级选项
  - HTTP/自定义元数据编辑
  - **分享链接** - 为文件创建安全的公开 URL，支持密码保护、过期时间和下载次数限制

- **👀 文件处理**
  - 浏览器内文件预览
    - PDF 文档
    - 图片
    - 文本文件
    - Markdown
    - CSV
    - Logpush 文件
  - 浏览器内文件编辑
  - 支持文件夹上传

- **📧 件集成**
  - 通过 Cloudflare Email Routing 接收和处理邮件
  - 直接在界面中查看邮件附件
